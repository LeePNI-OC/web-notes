# Linux 基础命令
* `pwd` 查看当前所在路径
* `ls` 查看当前目录文件 `-a` 查看所有 `-l` 查看详细信息
* `cd` 切换目录 `j` jump 跳转目录模糊查询跳转`j wwwroo__`
* `rm` 删除文件不能删除文件夹 `rm -r` 删除所有 `rm -rf` 删除所有忽略提示
* `/`根目录 `~`家目录 `./`当前目录 `../`上级目录
